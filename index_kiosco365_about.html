Acerca de esta herramienta

Esta herramienta fue desarrollada con el objetivo de optimizar tareas laborales, reducir el uso de papel y centralizar la información mediante formularios digitales y generación de documentos PDF.

Está diseñada para funcionar principalmente de manera offline, garantizando continuidad de uso incluso sin conexión a internet, aunque también admite funcionamiento online cuando está disponible.

https://filebeam.xyz

Tecnologías utilizadas

• HTML, CSS y JavaScript para la estructura y lógica general de la herramienta.

• Bootstrap para el diseño visual y la adaptación responsiva de la interfaz.

• jsPDF (integrado localmente) para la generación de documentos PDF de forma completamente offline, sin depender de servicios externos.

• IndexedDB para el almacenamiento local de datos del formulario, permitiendo conservar la información ingresada incluso ante cortes de energía o cierres inesperados del navegador.

• Firebase Realtime Database para el almacenamiento y sincronización de ciertos datos cuando se dispone de conexión a internet.

Almacenamiento de PDFs

Los documentos PDF generados pueden almacenarse mediante una API de almacenamiento en la nube llamada FileBeam.

La API recibe archivos mediante solicitudes POST multipart/form-data, permitiendo subir archivos de hasta 200 MB o archivos temporales de hasta 1 GB.

Para archivos temporales, se pueden definir tiempos de expiración como: 5 minutos, 30 minutos, 1 hora, 6 horas, 12 horas o 24 horas.

Documentación oficial:
https://github.com/filebeam/api
